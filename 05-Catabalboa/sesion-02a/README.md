<p>En esta clase se habló sobre las resistencias, el por qué tiene colores y que significan estos y en que se diferencian. Y con esto podemos saber cuanto equivale la resistencia, tambien hay paginas o una app en concreto con la que podemos calcular esto con mas facilidad, la aplicación se llama **Electrodoc**</p>
